# Zeitung

Final degree project, using PHP, JavaScript, HTML, CSS, and MySQL, CodeIgniter, JQuery, Chart.js, and Bootstrap.

## Release Information

## Server Requirements

PHP version 5.6 or newer is recommended.

## Starting the Project

```bash
docker-compose up
```

Then navigate to http://localhost:8080
